### About QuizApp
AnimQuizApp, also known as QuizApp, is an Educational app that quizzes a user about animals. 
Written in java and XML:
Questions are numbered from 1 to 6:
RadioGroup
RadioButton
RelativeLayout
CheckBox
Button
scores are shown with Toast
